# 02. 大模型应用开发范式

## 复习抓手

- 纯 Prompt：适合任务简单、上下文较短、无需外部工具的场景。
- Agent + Function Calling：适合需要工具调用、动态决策和多步执行的场景。
- RAG：适合需要接入私有知识、降低幻觉、增强可追溯性的场景。
- Fine-tuning：适合稳定任务风格、领域表达或格式高度固定的场景。
- 选择顺序通常是：Prompt 优先，知识不足用 RAG，需要行动用 Agent，稳定高频任务再考虑微调。
