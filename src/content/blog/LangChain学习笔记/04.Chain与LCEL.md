## create_sql_query_chain、create_stuff_documents_chain

### create_sql_query_chain

![image.png](figs/image%2062.png)

```python
from langchain_openai import ChatOpenAI
from langchain.chains import create_sql_query_chain
from langchain_community.utilities import SQLDatabase

#连接数据库
#mysql+pymysql://用户名:密码@ip地址:端口号/数据库名
db = SQLDatabase.from_uri("mysql+pymysql://用户名:密码@ip地址:端口号/数据库名")
#调用大模型
llm = ChatOpenAI(model="gpt-3.5-turbo", temperature=0)
#创建create_sql_query_chain实例
chain = create_sql_query_chain(llm,db)
#question是固定的，不能更改，返回SQL语句
respone=chain.invoke(
		{"question":"How many employees are there"}，
		{"table_name_to_use":["employees"]}
)
```
