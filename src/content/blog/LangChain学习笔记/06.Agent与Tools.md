# 06. Agent与Tools

## 复习抓手

- Tool 是 Agent 能执行动作的接口，核心属性包括 name、description、args_schema、func。
- Agent = LLM + Memory + Tools + Planning + Action。
- AgentExecutor 负责调度 Agent 的思考、工具调用和结果返回。
- ReAct 适合展示推理-行动过程，Function Calling 适合结构化工具调用。
- Agent 接入 Memory 时，重点看 history 变量如何传入提示词和执行器。
