# 07. Retrieval与RAG

## 复习抓手

- RAG 的主线：加载文档 -> 文档拆分 -> 向量化 -> 向量存储 -> 检索 -> 组装上下文 -> 调用模型。
- Document Loaders 负责把不同格式文件转成 Document。
- Text Splitters 负责控制 chunk 大小、重叠和语义完整性。
- EmbeddingModel 把文本转成向量，VectorStore 负责存储和相似性搜索。
- Retriever 是面向 Chain/Agent 的统一检索接口，重点掌握 similarity、score_threshold、MMR。

## TextLoader、PyPDFLoader、CSVLoader、JSONLoader

### txt文档：TextLoader

1、指明txt文档路径

```python
file_path="文档路径.txt"
```

2、创建一个TextLoader的实例

```python
text_loader = TextLoader(
		file_path = file_path,
		encoding = "utf-8" #指明文档的字符集
		
)
```

3、调用loader()，返回list[Docunment]

```python
docs = text_loader.load()
```

### json文档：JSONLoader

1.安装依赖包

```python
pip install jq
```

2、指明json文档路径

```python
file_path="文档路径.json"
```

3、加载json中所有数据

```python
json_loader = JSONLoader(
		file_path = file_path,	
		jq_schema=".", #表示加载所有字段
		text_content=False, #将加载的json对象转换成json字符串
)
```

4、加载json文件中的xxx[]中的所有yyy字段

```python
json_loader = JSONLoader(
		file_path = file_path,	
		jq_schema=".xxx[].yyy", #加载json文件中的xxx[]中的yyy字段
		#text_content=False, 不需要text_content属性，返回的是json字符串
)
```

5、加载json文件中嵌套在nnn.xxx[].yyy中的文本

```python
#方式1
json_loader = JSONLoader(
		file_path = file_path,	
		jq_schema=".nnn.xxx[].yyy", #加载json文件中嵌套在nnn.xxx[].yyy中的文本
)

#方式2
json_loader = JSONLoader(
		file_path = file_path,	
		jq_schema=".nnn.xxx[]", 
		content_key=".yyy",
		is_content_key_jq_parsable=True,
)
```

6、加载json文件中嵌套在nnn.xxx[].yyy和nnn.xxx[].zzz中的文本

```python
json_loader = JSONLoader(
		file_path = file_path,	
		jq_schema=".nnn.xxx[]", 
		content_key='.yyy + "\\n\\n" + .zzz',
		is_content_key_jq_parsable=True,
)
```
