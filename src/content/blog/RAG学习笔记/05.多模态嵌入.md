# 多模态嵌入

这篇内容主要讲的是：**如何把文本、图像等不同类型的数据，映射到同一个向量空间中，从而实现跨模态检索与理解**。

# CLIP 模型

CLIP 训练时使用对比学习（Contrastive Learning，对比学习）。具体来说，模型会拉近正确图文对的向量距离，同时推远错误图文对的向量距离。也就是常说的：**拉近正样本，推远负样本**。

| 组成 | 作用 |
| --- | --- |
| 图像编码器 | 把图片编码成图像向量 |
| 文本编码器 | 把文本编码成文本向量 |
| 共享向量空间 | 让图像向量和文本向量可以比较 |
| 相似度计算 | 判断图文是否匹配 |

# bge-visualized-m3

| 特性 | 中文解释 | 作用 |
| --- | --- | --- |
| Multi-Linguality（多语言性） | 支持超过 100 种语言 | 可用于跨语言文本或图文检索 |
| Multi-Functionality（多功能性） | 支持不同检索范式 | 可用于密集检索、多向量检索等 |
| Multi-Granularity（多粒度性） | 支持短文本到长文档 | 文本侧最长可处理 8192 个 token |
